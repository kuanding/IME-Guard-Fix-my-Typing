# IME Guard: Fix my Typing

[English](README.md) | [繁體中文](README.zh-TW.md) | [简体中文](README.zh-CN.md) | [日本語](README.ja.md) | [한국어](README.ko.md) | [Tiếng Việt](README.vi.md) | [हिन्दी](README.hi.md) | [தமிழ்](README.ta.md) | [ไทย](README.th.md) | [हिन्दी](README.hi.md) | [தமிழ்](README.ta.md) | [ไทย](README.th.md)

IME Guard: Fix my Typing என்பது Facebook, Gemini, Slack மற்றும் Discord போன்ற தளங்களில் IME (தமிழ், இந்தி போன்ற) மூலம் தட்டச்சு செய்யும் போது `Enter` விசையை அழுத்துவதால் செய்திகள் தவறுதலாக அனுப்பப்படும் WebKit பிழையை சரிசெய்ய வடிவமைக்கப்பட்ட ஒரு Safari நீட்டிப்பு (Extension) ஆகும்.

நீங்கள் தமிழ் போன்ற IME ஐச் சார்ந்த எந்த மொழியிலும் தட்டச்சு செய்தால், எழுத்துத் தேர்வை உறுதிசெய்ய `Enter` அழுத்தும் போது உங்கள் செய்தி தவறுதலாக அனுப்பப்படும் சிக்கலை நீங்கள் சந்தித்திருக்கலாம். இந்த நீட்டிப்பு அதைச் சரிசெய்கிறது!

## 🌟 அம்சங்கள்

- **தனிப்பயன் அனுமதிப்பட்டியல்**: இந்தச் சிக்கல் ஏற்படும் டொமைன்களைச் சேர்க்கவும் (எ.கா. `facebook.com` அல்லது `gemini.google.com`).
- **வைல்டு கார்டு ஆதரவு**: அனைத்து துணை டொமைன்களையும் உள்ளடக்க `*.facebook.com` என்பதைப் பயன்படுத்தவும்.
- **ஸ்மார்ட் இடைமறிப்பு**: நீங்கள் IME ஐப் பயன்படுத்தி தட்டச்சு செய்யும் போது மட்டுமே `Enter` விசை தடுக்கப்படும். வழக்கமான `Enter` அழுத்தங்கள் இன்னும் சரியாக வேலை செய்கின்றன.
- **பிரீமியம் டார்க் மோட் UI**: ஒரு நவீன உள்ளமைவு இடைமுகம்.
- **பல்மொழி ஆதரவு**: உங்கள் கணினியின் மொழியின் அடிப்படையில் தானாகவே மொழியை மாற்றுகிறது.

## 🚀 நிறுவல்

இந்த நீட்டிப்பு இன்னும் Mac App Store இல் கிடைக்கவில்லை. நீங்கள் Xcode ஐப் பயன்படுத்தி இதைத் தொகுத்து நிறுவலாம்:

1. உங்கள் Mac இல் [Xcode](https://developer.apple.com/xcode/) நிறுவப்பட்டுள்ளதா என்பதை உறுதிப்படுத்தவும்.
2. இந்த ரெபோசிட்டரியை (Repository) உங்கள் கணினியில் குளோன் (Clone) செய்யவும்.
3. Terminal ஐத் திறந்து திட்ட கோப்பகத்திற்கு (Project Directory) செல்லவும்.
4. இதை Safari நீட்டிப்பு திட்டமாக மாற்ற பின்வரும் கட்டளையை இயக்கவும்:
   ```bash
   xcrun safari-web-extension-converter .
   ```
5. Xcode தானாகவே மாற்றப்பட்ட திட்டத்தைத் திறக்கும். இதை உருவாக்க மற்றும் இயக்க மேல் இடதுபுறத்தில் உள்ள **Run** பொத்தானைக் கிளிக் செய்யவும் (அல்லது `Cmd + R` ஐ அழுத்தவும்).
6. Safari ஐத் திறந்து, **Settings > Extensions** க்குச் சென்று, இதை இயக்க **IME Guard: Fix my Typing** க்கான பெட்டியை சரிபார்க்கவும்.

## 💡 பயன்படுத்துவது எப்படி

1. "premature enter" சிக்கல் உள்ள இணையதளத்தைப் பார்வையிடும்போது (எ.கா. Facebook).
2. உங்கள் Safari கருவிப்பட்டியில் உள்ள **IME Guard: Fix my Typing** ஐகானைக் கிளிக் செய்யவும்.
3. **இந்த தளத்தைப் பாதுகாக்கவும் (Protect Current Site)** என்பதைக் கிளிக் செய்யவும். ஐகான் பேட்ஜ் பச்சை நிற **ON** ஐக் காண்பிக்கும்.
4. இப்போது நீங்கள் தட்டச்சு செய்ய உங்கள் IME ஐப் பயன்படுத்தலாம் மற்றும் எழுத்துக்களை உறுதிப்படுத்த `Enter` ஐ அழுத்தலாம். செய்தி இனி தவறுதலாக அனுப்பப்படாது!
