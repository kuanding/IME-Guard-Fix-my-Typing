# IME Guard: Fix my Typing

IME Guard: Fix my Typing เป็นส่วนขยายสำหรับ Safari ที่ออกแบบมาเพื่อแก้ไขข้อบกพร่องทั่วไปของ WebKit โดยที่การกดปุ่ม `Enter` ในขณะที่ใช้ Input Method Editor (IME) เช่น ภาษาไทย จะส่งข้อความก่อนกำหนดบนไซต์อย่าง Facebook, Gemini, Slack และ Discord

หากคุณพิมพ์ภาษาไทย จีน ญี่ปุ่น หรือภาษาใดๆ ที่ต้องใช้ IME คุณอาจพบปัญหาที่น่ารำคาญซึ่งการยืนยันการเลือกตัวอักษรของคุณด้วย `Enter` ทำให้ส่งข้อความที่ยังไม่เสร็จโดยไม่ได้ตั้งใจ ส่วนขยายนี้จะแก้ไขปัญหานั้น!

## 🌟 คุณสมบัติ

- **รายการที่อนุญาตที่กำหนดเอง**: เพิ่มโดเมนที่เกิดปัญหานี้ (เช่น `facebook.com` หรือ `gemini.google.com`)
- **รองรับไวลด์การ์ด**: ใช้ `*.facebook.com` เพื่อครอบคลุมโดเมนย่อยทั้งหมด
- **การสกัดกั้นอัจฉริยะ**: ปุ่ม `Enter` จะถูกสกัดกั้นเฉพาะในขณะที่คุณกำลังพิมพ์ข้อความโดยใช้ IME เท่านั้น การกด `Enter` แบบปกติยังคงใช้งานได้ดี
- **UI โหมดมืดระดับพรีเมียม**: อินเทอร์เฟซการกำหนดค่าที่ทันสมัยและโฉบเฉี่ยว
- **รองรับหลายภาษา**: สลับภาษาอัตโนมัติตามภาษาของระบบของคุณ

## 🚀 การติดตั้ง

ส่วนขยายนี้ยังไม่มีให้บริการบน Mac App Store คุณสามารถคอมไพล์และติดตั้งได้ด้วยตนเองโดยใช้ Xcode:

1. ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง [Xcode](https://developer.apple.com/xcode/) บน Mac ของคุณ
2. คัดลอก (Clone) ที่เก็บนี้ไปยังเครื่องคอมพิวเตอร์ของคุณ
3. เปิด Terminal และนำทางไปยังไดเรกทอรีของโครงการ
4. เรียกใช้คำสั่งต่อไปนี้เพื่อแปลงเป็นโครงการส่วนขยาย Safari:
   ```bash
   xcrun safari-web-extension-converter .
   ```
5. Xcode จะเปิดโครงการที่แปลงแล้วโดยอัตโนมัติ คลิกปุ่ม **Run** ที่มุมบนซ้าย (หรือกด `Cmd + R`) เพื่อสร้างและรัน
6. เปิด Safari ไปที่ **การตั้งค่า (Settings) > ส่วนขยาย (Extensions)** และทำเครื่องหมายที่ช่องสำหรับ **IME Guard: Fix my Typing** เพื่อเปิดใช้งาน

## 💡 วิธีใช้งาน

1. เมื่อคุณเยี่ยมชมเว็บไซต์ที่มีปัญหา "ส่งข้อความก่อนกำหนด" (เช่น Facebook)
2. คลิกที่ไอคอน **IME Guard: Fix my Typing** ในแถบเครื่องมือ Safari ของคุณ
3. คลิก **ปกป้องไซต์นี้ (Protect Current Site)** ป้ายไอคอนจะแสดงสีเขียว **ON**
4. ตอนนี้คุณสามารถใช้ IME ของคุณเพื่อพิมพ์และกด `Enter` เพื่อยืนยันตัวอักษร ข้อความจะไม่ถูกส่งก่อนกำหนดอีกต่อไป!
